About

This application is a simple TODO list built with Angular and Angular Material. It allows you to create, edit, complete, and delete tasks.

Features

  • View a list of tasks with optional descriptions
  • Add new tasks using a dialog with validation
  • Edit existing tasks in the same dialog
  • Mark tasks as complete or incomplete
  • Filter tasks using the “Show all” option
  • Delete tasks from the list
